一鍵抓名單

第一次先安裝一次

1

下載並解壓縮

  1. 按上方藍色按鈕下載 ZIP。
  2. 下載完成後，對 ZIP 按兩下解壓縮。
  3. 看到一個工具資料夾就對了，資料夾裡面會有 manifest.json。
2

載入到 Chrome

  1. 用 Chrome 打開 chrome://extensions/。
  2. 開啟右上角「開發人員模式」。
  3. 開啟後，頁面左上角會出現「載入未封裝項目」按鈕，點它。
  4. 選擇剛剛解壓縮出來的工具資料夾。

每次要抓名單就這樣用

1

打開 WhatsApp Web

進入聊天列表、聯絡人列表，或群組成員列表。

2

點右上角工具圖示

點 Chrome 右上角的「成交聯盟 WhatsApp 抓名單」圖示，WhatsApp 右上角會出現面板。

3

下載 CSV

按「一鍵抓名單」，等完成後按「下載 CSV」。下載的檔案會在電腦的「下載項目」。

卡住時看這裡

看不到右上角工具圖示

點 Chrome 右上角拼圖圖示，把「成交聯盟 WhatsApp 抓名單」固定到工具列。

點了沒有出現面板

先確認目前頁面是 web.whatsapp.com，重新整理 WhatsApp Web 後再點一次。

找不到「載入未封裝項目」

先確認右上角「開發人員模式」已打開；打開後，按鈕會出現在頁面左上角。

Chrome 說不能選 ZIP

不要選 ZIP 檔，要先解壓縮，再選解壓縮後出現的工具資料夾。

CSV 下載後要做什麼

把 CSV 丟進原本的「名單轉通訊錄 CSV 工具」即可。

七天試用

第一次在 WhatsApp Web 使用時開始計算七天。到期後，工具會停止抓名單與下載 CSV。

下載後怎麼做

把下載的 CSV 丟進原本的名單轉通訊錄工具即可。
