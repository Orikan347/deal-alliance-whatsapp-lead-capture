待會照這樣操作

1

下載並解壓縮

下載 Chrome 擴充元件 ZIP，解壓縮到電腦裡。

2

載入擴充元件

打開 chrome://extensions/，開啟「開發人員模式」，點「載入未封裝項目」，選擇解壓縮後的資料夾。

3

一鍵抓名單

打開 WhatsApp Web，點 Chrome 右上角工具圖示。面板出現後按「一鍵抓名單」，完成後按「下載 CSV」。

七天試用

第一次在 WhatsApp Web 使用時開始計算七天。到期後，工具會停止抓名單與下載 CSV。

下載後怎麼做

把下載的 CSV 丟進原本的名單轉通訊錄工具即可。

備用書籤方式

如果不能安裝擴充元件，才使用這段備用書籤程式碼。書籤方式較容易受 Chrome 或 WhatsApp Web 影響，不建議當主流程。
